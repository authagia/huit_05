## 🛒 【タスク1】自動化アイテムの実装（作業時間：25分）

前回作った「連打ゲーム」に、放置していても勝手にコインが増える「自動化アイテム（ショップ）」を組み込みます。条件分岐（`if`）とタイマー処理（`setInterval`）を学びます。

### ✍️ 実装手順（ロジックの追加）

HTMLにショップ用のボタンを追加し、JavaScriptに以下の購入ロジックとタイマーを追加します。

```javascript
let autoCPS = Number(localStorage.getItem("savedAutoCPS")) || 0; // 1秒あたりの自動増加量
let itemCost = 10; // アイテムの価格

// 【アイテム購入処理】
upgradeBtn.addEventListener("click", () => {
    // コインが足りている時だけ買える（条件分岐）
    if (coin >= itemCost) {
        coin = coin - itemCost; // コインを消費
        autoCPS = autoCPS + 1;  // 自動パワーを+1

        // 画面と金庫を更新
        coinCountEl.textContent = coin;
        autoCountEl.textContent = autoCPS;
        localStorage.setItem("savedCoin", coin);
        localStorage.setItem("savedAutoCPS", autoCPS);
    }
});

// 【タイマー処理】1秒（1000ミリ秒）ごとに勝手に増える
setInterval(() => {
    if (autoCPS > 0) {
        coin = coin + autoCPS;
        coinCountEl.textContent = coin;
        localStorage.setItem("savedCoin", coin);
    }
}, 1000);

```

これが動けば、夢の「放置ゲー」が完成です！**スプシの「1. 自動化完了」に `◯` をつけましょう！**

## 🏆 【伏線回収】超・発展エクストラクエスト

早く終わった自走勢のためのチャレンジ課題です。第4回で張った伏線を回収してみよう！

### 🦖 クエスト1：処理の「関数化」（コードをスマートにする）

クリックした時と、タイマーで増える時、どちらも「`coin` を増やして `coinCountEl.textContent` を変える」という同じコードを書いていますよね。これを `function plusCoin() { ... }` という一つの関数にまとめて、両方から呼び出すように書き換えてみよう！

### 🔒 クエスト2：強いアイテムの「アンロック機能」

HTMLに、コスト100枚の超強力な「おばあちゃんボタン」を新しく作ります。

1. CSSで最初は `display: none;` にして画面から隠す。
2. クリックした時やタイマーが動くたびに `if (coin >= 100)` をチェックする。
3. 100枚を超えた瞬間、ボタンのスタイルを `display: block;` に変えて、画面に出現させてみよう！
