第4回 講師向けメモ

目的
- 授業で参加者に配布する虫食いテンプレートと、講師用解答を分けて管理する。

配布物（参加者）
- `10_テンプレート.html`
- `11_配布用_虫食い.js`
- `README_PARTICIPANT.md`

講師専用（非公開）
- `10_テンプレート_answer.html`
- `11_サンプルコード.js`
- `README_INSTRUCTOR.md` (本ファイル)

授業の流れ（推奨）
1. 参加者に配布物を配る。
2. 各自で `10_テンプレート.html` をブラウザで開かせ、動作を確認させる（クリック → リロードで消える体験）。
3. `11_配布用_虫食い.js` の TODO を埋めさせる（所要約45〜60分）。
4. 実装後、講師は `10_テンプレート_answer.html` を提示して完成版を見せ、実装のポイントを解説する。

講師のデモ用チェック
- 事前に `10_テンプレート_answer.html` をローカルで開いて動作確認済みにする。
- `11_サンプルコード.js` の `itemCost` や `autoCPS` の初期値は授業の難易度に合わせて調整可能。

よくある問題と対処
- localStorage の parse エラー → 受講者にブラウザの開発者ツールで `Application > Local Storage` から `myGameData` を削除させる。
- 数字が増えない/表示されない → HTML 中の `id` 名が正しいか (`coinCount`, `clickButton`, `upgrade-btn`, `auto-count`, `playerName`) を確認。
- 保存が効かない → ブラウザのプライベートウィンドウや拡張による干渉を確認。

配布方法の注意
- 配布用ファイルには解答コードを含めないこと（`10_テンプレート.html` は `11_配布用_虫食い.js` を読み込む）。

ファイル場所
- すべてのファイルは `第4回資料/` にあります。

希望があれば、講師用の短いスライドや解説スニペットを追加で作成します。
