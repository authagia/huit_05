# トラブルシューティング

## 1. 「クリックしても増えない」
- `id` が HTML と JS で一致しているか確認する
- `addEventListener` を正しい要素に設定しているか確認する
- `clickCount` に `let` で宣言があるか確認する

## 2. 「保存しても復元されない」
- `localStorage.setItem` のキー名が誤っていないか
- `JSON.stringify` / `JSON.parse` の使い方が正しいか
- `loadGame()` をページ読み込み時に呼び出しているか
- `localStorage` の保存前に `alert` などで早めに確認してみる

## 3. 「リロードしたら0に戻る」
- `let coin = 0;` だけになっていないか
- `Number(localStorage.getItem(...)) || 0` の形で読み込んでいるか
- `coinCountEl.textContent = coin;` を読み込み後に実行しているか

## 4. 「100枚でアンロックしない」
- `checkUnlock()` が `plusCoin()` の中で呼ばれているか
- `unlockArea` の `id` が一致しているか
- `style.display` の値が文字列で指定されているか（例: `'block'`）

## 5. 「自動化アイテムが動かない」
- `setInterval()` の秒数を 1000 にしているか
- `autoCPS` が `0` のままになっていないか
- `coin` を増やした後に `updateDisplay()` や `saveGame()` を呼び出しているか

## 6. 「アニメーションが動かない」
- CSSに `transition` や `:active` を正しく書いているか
- 該当要素にクラス名やIDが付いているか
- ブラウザのキャッシュをクリアして最新 CSS を読み込ませる

## 7. よくある修正例
- `document.getElementById('playerName').value` を `playerName` に代入していない
- `updateDisplay()` を呼び忘れている
- `JSON.parse` の前に `localStorage.getItem` が `null` になっていないかチェックしていない
