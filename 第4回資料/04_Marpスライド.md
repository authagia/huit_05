## イベントリスナーでボタンを「聞く」

### JavaScript: クリックされたときの処理

```javascript
// HTML要素を取得
const coinCountEl = document.querySelector("#coin-count");
const clickBtn = document.querySelector("#click-btn");

// 変数を作る
let coin = 0;

// クリックを「聞く」
clickBtn.addEventListener("click", () => {
    coin = coin + 1;                  // 変数を1増やす
    coinCountEl.textContent = coin;   // 画面に表示
});
```

### 🔍 動作確認

ブラウザでボタンをクリック → 数字が増える ✅

<!-- 
話す内容:
- `addEventListener` は「〇〇が起きたら、この関数を実行してね」という命令
- 矢印関数 `() => { }` はプログラマの「手順書」
-->

## localStorageの実装

### 1️⃣ クリック時に保存

```javascript
clickBtn.addEventListener("click", () => {
    coin = coin + 1;
    coinCountEl.textContent = coin;
    
    // 【追加】金庫に保存
    localStorage.setItem("savedCoin", coin);
});
```

### 2️⃣ ページ起動時に読み込む

```javascript
// 修正: let coin = 0; を以下に置き換え
let coin = Number(localStorage.getItem("savedCoin")) || 0;

// 読み込んだ値を画面に表示
coinCountEl.textContent = coin;
```

<!-- 
話す内容:
- `Number(...)` は「文字を数字に変換」する魔法
- `|| 0` は「空だったら0にする」という意味
-->

## 🔑 発展課題：アンロック機能

### 隠されたボタンを出現させる

#### CSSで最初は隠す

```css
#secret-btn {
    display: none;  /* 画面から隠す */
}
```

#### JSで100枚になったら表示

```javascript
if (coin >= 100) {
    document.querySelector("#secret-btn").style.display = "block";
}
```

**データの変化 = UI変化** → 動的なゲーム性！
