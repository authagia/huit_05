第4回 配布物（参加者向け）

目的
- このテンプレートは演習用の虫食いコードを含みます。手を動かして実装部分を埋めてください。

ファイル
- `10_テンプレート.html` : ブラウザで開いて使う配布HTML（虫食いテンプレート）
- `11_配布用_虫食い.js` : 実装してもらうJSの骨組み（TODO コメントあり）
- `11_サンプルコード.js` : 講師用の完成版サンプル（参照用）

進め方
1. `10_テンプレート.html` をブラウザで開いて現在の動作を確認する。まずはクリックで何が起きるか観察。
2. `11_配布用_虫食い.js` の TODO 部分を埋める。
   - `checkUnlock()`
   - `plusCoin()`
   - `saveGame()`
   - `loadGame()`
3. 実装後、ブラウザをリロードして保存・復元が動くか確認する。
4. 発展課題: 自動化アイテム（autoCPS）やアニメーションを追加してみる。

ヒント
- localStorage に保存する際は `JSON.stringify` を使うと複数の値が扱いやすくなります。
- 読み込み時は `JSON.parse` と存在チェックを忘れないでください。

質問があれば講師に聞いてください。
