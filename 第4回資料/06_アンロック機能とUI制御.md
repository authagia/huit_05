# アンロック機能と UI 制御

## 1. アンロック機能の目的
- 一定のコイン数に到達したときに、新しいボタンやメッセージを表示する
- イベントに段階をつけて、ゲーム感を出す

## 2. 100枚でボタンを出す例
```js
function checkUnlock() {
  const unlockArea = document.getElementById('unlockArea');
  if (clickCount >= 100) {
    unlockArea.style.display = 'block';
  } else {
    unlockArea.style.display = 'none';
  }
}
```

## 3. UI 表示切り替えの方法
- `style.display = 'none'` : 非表示
- `style.display = 'block'` : 表示
- CSS クラスを使って見た目を変更することもできる

## 4. アンロック後の演出例
- `20%` の「特別メッセージ」
- `スペシャルアイテムを獲得` ボタンを表示
- `clickCount` に応じて異なる条件分岐を追加

```js
if (clickCount >= 100) {
  unlockArea.style.display = 'block';
  document.getElementById('unlockMessage').textContent = 'おめでとう！スペシャルアイテムを使えるようになりました！';
}
```

## 5. 追加の UI 制御例
- `disabled` 属性を切り替える
- ボタンのテキストを変更する
- 複数段階のアンロックを実装する
