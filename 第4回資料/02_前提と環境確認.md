# 前提と環境確認

## 1. 実施前に確認すること
- Webブラウザがインストールされていること（Chrome / Edge / Firefox など）
- テキストエディタが使えること（VS Code 推奨）
- 作業用フォルダが準備されていること
- 第4回で配布するテンプレートファイルがあること

## 2. フォルダ構成例
- `第4回資料/`
  - `10_テンプレート.html`
  - `11_サンプルコード.js`
  - `01_目次.md`
  - ...

## 3. 使う技術の一覧
- HTML: 入力欄、ボタン、表示エリア
- CSS: 最低限の見た目調整（今回はHTML内スタイルでも可）
- JavaScript: イベント、関数、変数、DOM操作
- `localStorage`: データの保存と読み込み

## 4. 事前チェックリスト
- [ ] テンプレートを開いて画面が表示される
- [ ] `playerName` 入力欄がある
- [ ] 「クリック！」 ボタンがある
- [ ] `localStorage` の保存先が決まっている（キー名を確認）
