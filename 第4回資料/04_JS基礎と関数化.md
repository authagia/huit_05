# JS基礎と関数化

## 1. クリックでコインを増やす処理
- `button` のクリックイベントを使う
- `clickCount` という変数でコイン数を管理する
- ボタンを押したら `plusCoin()` を呼ぶ

## 2. 変数と関数の定義
```js
let playerName = '';
let clickCount = 0;

function plusCoin() {
  clickCount += 1;
  updateDisplay();
  checkUnlock();
}
```

## 3. 表示を更新する関数
```js
function updateDisplay() {
  document.getElementById('coinCount').textContent = clickCount;
}
```

## 4. 関数化のメリット
- 同じ処理を何度も書かなくて済む
- 画面の更新や判定を分かりやすく整理できる
- 別の場所からも同じ処理を呼び出せる

## 5. 実装のパーツ
このゲームを作るには、小さな機能を組み合わせます。スライドのように、システムを実現するための部品として考えるとわかりやすくなります。

- HTML要素の取得: 画面の部品を JS に繋げる
- クリックイベント: ボタンが押された瞬間を検知する
- 変数で状態を管理: コイン数やアイテムの値を保持する
- 画面更新: `textContent` で値を表示する
- 保存 / 読み込み: `localStorage` でリロード後も値を残す
- 条件判定: `if` でアンロックや購入をチェックする
- アイテム購入: コインを消費して性能を上げる
- アンロック表示: 条件を満たしたときに要素を出す
- アニメーション: CSS で動きをつける

## 6. クリック処理の接続
```js
document.getElementById('clickButton').addEventListener('click', function() {
  plusCoin();
});
```

## 7. playerName の扱い
- 入力欄の値を取得して `playerName` に代入する
- 保存時に名前も一緒に保存する

```js
playerName = document.getElementById('playerName').value;
```
