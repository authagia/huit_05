# 発展課題（クエスト）

## 1. 追加チャレンジ例
- 150枚で「自動化アイテム（ショップ）」を買えるようにする
- 200枚で「アニメーション付きアンロックボタン」を出現させる
- 300枚で「称号表示」を追加
- `playerName` によって特別なメッセージを出す

## 2. 具体的な課題
- 課題 A: 100枚でボタン表示、200枚でメッセージ表示
- 課題 B: `clickCount` を 10 ずつ増やす「まとめクリック」機能を作成
- 課題 C: `localStorage` に保存したデータをページロード時に復元し、再度コイン数を表示する

## 3. 実装例
```js
function bonusClick() {
  clickCount += 10;
  updateDisplay();
  checkUnlock();
}
```

## 4. 発展アイデア
- `playerName` に応じて称号を変える
- 複数のアイテムを追加し、条件ごとに表示
- セーブ回数をカウントして表示する

## 5. フォローアップ
- 条件を増やすときは `if` の順番に注意
- `checkUnlock()` の中で複数の判定を整理して書くと見やすい

## 6. テストプレイ
- 自分が作ったゲームを他の人に見せて操作してもらう
- 友だちのゲームを遊んで、使いやすさ・見た目の違いを比べる
- 気に入った工夫を自分のゲームにも取り入れてみる
