# トラブルシューティング

## Q1: winget が見つからない

**解決**:
- Windows 10/11 の最新アップデートを適用
- または https://github.com/microsoft/winget-cli からインストール

---

## Q2: gh auth login でエラー

**解決**:
1. 既にログインしていないか確認: `gh auth status`
2. ログアウト后再度ログイン: `gh auth logout` → `gh auth login`

---

## Q3: git push で認証エラー

**解決**:
```bash
gh auth refresh -s repo
```

---

## Q4: VS Codeでターミナルが開けない

**解決**:
- `Ctrl + @` で開く
- メニュー: 「ターミナル」 → 「新しいターミナル」
- 上のメニューから「表示」→「ターミナル」

---

## Q5: GitHub Pagesが公開されない

**確認**:
1. リポジトリがPublicか（Privateだと公開されない）
2. `main` または `master` ブランチにpushしているか
3. ファイル名が `index.html` か

**解決**:
- Settings → Pages で Branch を確認
- index.html があることを確認

---

## Q6: 文字化けが発生する

**原因**: ファイルの文字コードがUTF-8ではない

**解決**:
1. VS Code で右下の「UTF-8」をクリック
2. 「Save with Encoding」→「UTF-8」で保存し直し

---

## Q7: リポジトリを削除したい

**解決**:
```bash
gh repo delete ユーザー名/リポジトリ名
```

または：
1. GitHubのリポジトリページ → 「Settings」
2. 一番下までスクロール → 「Delete this repository」
