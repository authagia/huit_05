# Git インストールと初期設定

## Windowsの場合（wingetを使用)

### 1. インストール

PowerShell またはコマンドプロンプトで以下を実行:

```powershell
winget install Git.Git
```

---

## Macの場合

### 1. インストール

1. ターミナルを開く（Cmd + Space → "ターミナル"）
2. 以下のコマンドを実行:

```bash
xcode-select --install
```

3. 「インストール」をクリック

---

## 確認チェック

- [ ] `git --version` でバージョンが表示される
