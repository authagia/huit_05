# VS Code インストールガイド

## Windowsの場合（wingetを使用)

### 1. インストール

PowerShellで以下を実行:

```powershell
winget install Microsoft.VisualStudioCode
```

### 2. 起動確認

1. スタートメニューから「Visual Studio Code」を検索
2. クリックして起動すればOK

---

## Macの場合

### 1. ダウンロード

1. ブラウザで https://code.visualstudio.com/ を開く
2. 「Mac」ボタンをクリックしてダウンロード

### 2. インストール

1. ダウンロードした `.zip` をダブルクリックして展開
2. 展開された「Visual Studio Code.app」を「アプリケーション」フォルダにドラッグ

### 3. 起動確認

1. Launchpad または Spotlight から「Visual Studio Code」を検索
2. クリックして起動すればOK

---
